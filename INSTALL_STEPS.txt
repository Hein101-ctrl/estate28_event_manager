ESTATE28 EVENT MANAGER V3 - UPDATE STEPS

THIS IS AN UPDATE PACKAGE FOR USERS ALREADY ON V1 OR V2.

1. Use the SAME GitHub repository already connected to Frappe Cloud.
2. Replace the repo files with this v3 package.
3. Commit and push to GitHub.
4. In Frappe Cloud, open the Bench Group and deploy the app update to your site.
5. After update, open Estate 28 Event and confirm these new fields exist:
   - Venue Area
   - Deposit Amount
   - Amount Paid
   - Balance Due
   - Internal Status

NOTES
- This package assumes you did NOT add manual Customize Form fields outside the app.
- Because your current setup came from the app itself, this is the correct clean update route.
- Balance Due auto-calculates from Package Amount minus Amount Paid.
